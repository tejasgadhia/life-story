# Life Story

Birthday analysis webapp - enter a birthdate, get a personalized historical/generational report.

## Live Demo

https://tejasgadhia.github.io/life-story/

## Features

- 12-section personalized report based on birth year
- Three visual themes: Timeline, Newspaper (1884-style), Case File (FBI dossier)
- Birthday commonality ranking (1-366)
- Generational identity analysis
- Historical milestones mapped to your age
- Stateful URLs for bookmarking/sharing

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router (client-side routing)
- GitHub Pages deployment

## Data Sources

- Birthday frequency: FiveThirtyEight (CDC/SSA 1994-2014)
- Generational definitions: Pew Research Center

## Local Development

```bash
npm install
npm run dev
```
