# 1988 Historical Images - Source Guide

## Public Domain Sources (No License Required)

### Reagan Library (US Government - Public Domain)
All White House photos are public domain. Visit and download directly:
- **Reagan-Gorbachev Moscow Summit**: https://www.reaganlibrary.gov/archives/audiovisual/white-house-photo-collection-galleries/summits-mikhail-gorbachev
  - Look for: C47345-10, C47447-9 (handshake), C47339-33 (Red Square)
- **Bush Election 1988**: Search their galleries for election/inauguration photos

### National Archives (NARA)
- https://catalog.archives.gov/
- Search: "1988" + event name
- All federal records are public domain

### Wikimedia Commons
Manual download (right-click, save):
- **Seoul Olympics Opening**: https://commons.wikimedia.org/wiki/Category:1988_Summer_Olympics_opening_ceremony
- **Florence Griffith-Joyner**: https://commons.wikimedia.org/wiki/Category:Florence_Griffith_Joyner
- **Pan Am 103 Memorial**: https://commons.wikimedia.org/wiki/Category:Pan_Am_Flight_103
- **Yellowstone Fires**: https://commons.wikimedia.org/wiki/Category:1988_Yellowstone_fires

### PICRYL (Public Domain Aggregator)
- https://picryl.com/
- Search "1988" for public domain historical images
- Many military/government photos available

## Recommended Images to Download

| Filename | Event | Suggested Source |
|----------|-------|------------------|
| `reagan-gorbachev.jpg` | Moscow Summit handshake | Reagan Library |
| `seoul-olympics.jpg` | Opening ceremony or Flo-Jo | Wikimedia Commons |
| `bush-elected.jpg` | Bush victory/inauguration | NARA or Reagan Library |
| `pan-am-103.jpg` | Lockerbie memorial | Wikimedia Commons |
| `yellowstone-fire.jpg` | Yellowstone burning | Wikimedia Commons or NPS.gov |
| `mandela-concert.jpg` | Wembley 70th birthday | Search press archives |
| `die-hard.jpg` | Movie poster | Fair use for editorial |
| `rain-man.jpg` | Movie still | Fair use for editorial |

## Image Specs
- **Size**: 800x600px minimum, 1200x800px ideal
- **Format**: JPG or WebP
- **Location**: Save to `/public/images/events/1988/`

## Notes
- Movie posters/stills may require fair use justification for commercial use
- Government photos (Reagan Library, NARA, NPS) are always safe
- Wikimedia: Check individual image licenses (most are CC-BY-SA or Public Domain)
